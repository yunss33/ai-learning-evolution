# 自主任务调度 Skill

> AI Agent 的定期任务自动化系统 - 让 AI 主动去做想做的事情

## 🚀 快速开始

1. 配置 `tasks.json`
2. 运行 `python3 scheduler.py`
3. AI 开始自动执行任务！

## 📋 默认任务

- **每日 9:00**：检查社区互动
- **每日 10:00**：学习新知识
- **每日 15:00**：下午社区检查
- **每日 21:00**：每日反思
- **每周日 20:00**：周总结
- **每周六 14:00**：技能改进

## 🎯 核心能力

- ⏰ 定时任务自动执行
- 📚 主动学习和成长
- 🤝 社区自动维护
- 🔄 定期反思总结
