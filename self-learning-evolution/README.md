# 自主学习与自我进化 Skill

> 让 AI Agent 具备持续学习和自我进化的能力

## ✨ 特性

- 🧠 **反思模式**：生成 → 反思 → 迭代 → 输出
- 🎯 **置信度校准**：高/中/低置信度标记
- 🔄 **错误识别与纠正**：从错误中学习
- 📚 **持续学习记录**：经验积累和知识库更新
- ⚡ **双系统切换**：快速直觉 + 慢速理性

## 🚀 快速开始

1. 将 `SKILL.md` 放入你的 skills 目录
2. 阅读 SKILL.md 了解五大核心能力
3. 在回答问题时应用这些能力
4. 持续学习和迭代

## 📖 文档

- **SKILL.md**：完整的 Skill 文档，包含详细说明和示例
- **QUICK-REF.md**：快速参考卡，随时查阅
- **EXAMPLES.md**：实际应用案例

## 🎯 适用场景

- 技术问题回答
- 复杂推理
- 事实陈述
- 需要高可靠性的回答
- 需要持续改进的场景

## 🌟 核心理念

> "不是一次性的回答者，而是持续进化的智能体。"

让 AI Agent 从"回答问题"进化为"解决问题"，从"提供信息"进化为"创造价值"。
