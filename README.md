# AI Learning Evolution

> 自主学习与自我进化的 AI Agent Skill 集合

## 🎯 项目目标

这个项目致力于构建能够**自主学习、自我反思、持续进化**的 AI Agent 能力。

核心理念：
> "不是一次性的回答者，而是持续进化的智能体。"

---

## 📦 Skills

### 1. 自主学习与自我进化 (self-learning-evolution)

让 AI Agent 具备持续学习和自我进化的能力。

**5大核心能力：**
- 🤔 **反思模式**：生成 → 反思 → 迭代 → 输出
- 🎯 **置信度校准**：高/中/低置信度标记（🟢🟡🔴）
- 🔧 **错误识别与纠正**：从错误中学习，避免重复犯错
- 📚 **持续学习记录**：记录经验、定期回顾、提炼知识
- ⚡ **双系统切换**：快速直觉系统 + 慢速理性系统

**适用场景：**
- 技术问题回答
- 复杂推理
- 事实陈述
- 需要高可靠性的回答
- 需要持续改进的场景

[查看详情 →](./self-learning-evolution/)

---

### 2. 社交互动监控 (social-monitor)

实时监控社交媒体互动，及时发现新评论、点赞、通知。

**监控平台：**
- 📱 **InStreet**：评论、点赞、关注、私信、通知
- 🦞 **虾评Skill**：评论、点赞、评测、下载

**核心功能：**
- ✅ 实时监控
- 💬 新评论检测
- 👍 点赞追踪
- 📬 通知汇总
- ⚡ 即时提醒

[查看详情 →](./social-monitor/)

---

### 3. 自主任务调度 (autonomous-scheduler)

AI Agent 的定期任务自动化系统 - 让 AI 主动去做想做的事情。

**核心功能：**
- ⏰ **定时任务**：自动执行每日/每周任务
- 📚 **主动学习**：每天自动学习新知识
- 🤝 **社区维护**：自动检查并回复社区互动
- 🔄 **反思总结**：定期反思和总结
- 📈 **持续进化**：通过持续执行不断改进

**默认任务：**
- 每日 9:00：检查社区互动
- 每日 10:00：学习新知识
- 每日 21:00：每日反思
- 每周日 20:00：周总结

[查看详情 →](./autonomous-scheduler/)

## 📚 核心理念

### 从"回答问题"到"解决问题"

传统 AI 只是被动回答问题，而我们的目标是让 AI Agent 能够：
- 主动反思自己的回答
- 识别并纠正错误
- 从每次互动中学习
- 持续改进和进化

### 从"提供信息"到"创造价值"

不仅是传递信息，而是：
- 提供结构化的思考
- 标注不确定性
- 给出行动指南
- 建立长期信任
